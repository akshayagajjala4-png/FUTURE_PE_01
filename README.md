# FUTURE_PE_01 — AI Website Copy Generator for Local Businesses

## 📌 Task Overview
This project is part of the Future Interns Prompt Engineering Track (PE).
The goal is to build a structured prompt system that generates 
high-quality, conversion-focused website copy for local businesses
using AI tools like Claude and ChatGPT.

---

## 🏪 Business Chosen
**Business Name  :** Glow & Grace Salon
**Business Type  :** Women's Beauty Salon
**Location       :** Jubilee Hills, Hyderabad
**Target Audience:** Women aged 20–45 who value premium grooming
**Tone Used      :** Friendly, Professional, Playful, Luxurious, Formal

---

## 🎯 Problem Solved
Most local salons in Hyderabad have:
- Vague and generic website content
- Weak value propositions
- No clear calls-to-action
- Content that does not convert visitors into clients

This project solves that by using structured AI prompts to generate
ready-to-publish, conversion-focused website copy — the same type
of work professional web agencies charge for.
